# ExamPilot MVP

一个面向**英语四级 + 期末考**的 AI 备考学习系统最小可用版本。

它不是宏大的 Life OS，而是一个更实际的短期备考工具：

```text
输入目标和资料
→ 生成今日计划
→ 提取考点
→ 生成卡片
→ 自测题
→ 晚间复盘
→ 第二天继续
```

## 当前状态

这是一个纯前端静态 MVP，目前包含两个版本：

- **v1 (index.html)**：基本功能版本，可直接打开使用。
- **v2 (index_v2.html)**：在 v1 基础上新增了文件上传和错题本功能，并将科目输入的示例改为四级专项和期末科目示例。

两者都无需后端、无需数据库，打开对应 HTML 即可运行，数据保存在浏览器 `localStorage`。如需接入 AI 接口，可在设置中填写 OpenAI 兼容 API。

## 功能

- 今日计划生成
- 资料粘贴与考点提取
- 主动回忆卡片
- 简化版间隔重复调度
- 自测题生成与判题
- 晚间复盘
- 本地数据导出
- 可选 AI 接口设置
- **v2 新增：文件上传**（支持读取本地 txt/md/doc 等文本文件）并将其填入资料框
- **v2 新增：错题本**（记录错题题目、答案、分析与标签，可清空）
- **v2 示例科目**：输入框示例更新为“四级单词、四级阅读、四级作文、四级翻译、期末科目”等

## 学习理论

内置的设计原则：

1. **主动回忆**：先测试，再看答案。
2. **间隔重复**：根据记忆情况安排下次复习。
3. **费曼解释**：要求用户用自己的话解释。
4. **错题归因**：复盘卡住的原因，而不是只记答案。
5. **最小行动**：把学习任务压缩成今天能做的小任务。

## 快速开始

### 方法一：直接打开

双击打开：

```text
index.html         # v1 基础版
index_v2.html      # v2 增强版（推荐）
```

### 方法二：用本地服务器

```bash
python -m http.server 5173
```

浏览器打开：

```text
http://localhost:5173/index.html
```

或：

```text
http://localhost:5173/index_v2.html
```

## GitHub Pages 部署

推送后可在 GitHub 仓库：

```text
Settings → Pages → Deploy from a branch → main → /root
```

然后访问 GitHub Pages 地址。

## AI 设置说明

设置页支持 OpenAI 兼容接口：

- `API Endpoint`
- `模型名`
- `API Key`

示例：

```text
https://api.openai.com/v1/chat/completions
```

注意：

> 纯前端无法真正隐藏 API Key。正式项目应改成“前端 → 后端 → AI API”。
